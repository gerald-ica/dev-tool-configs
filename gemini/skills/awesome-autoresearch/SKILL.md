# Awesome Autoresearch

A curated, high-signal index of autonomous improvement loops, research agents, and descendants inspired by [karpathy/autoresearch](https://github.com/karpathy/autoresearch).

## Categories

### General-Purpose Descendants
Frameworks that generalize autoresearch to any measurable metric:

| Tool | Description |
|------|-------------|
| [kayba-ai/recursive-improve](https://github.com/kayba-ai/recursive-improve) | Recursive self-improvement with execution traces and keep-or-revert evaluation |
| [uditgoenka/autoresearch](https://github.com/uditgoenka/autoresearch) | Claude Code skill generalizing autoresearch for software, docs, security, debugging |
| [supratikpm/gemini-autoresearch](https://github.com/supratikpm/gemini-autoresearch) | Gemini CLI skill with Google Search grounding and 1M token context |
| [zkarimi22/autoresearch-anything](https://github.com/zkarimi22/autoresearch-anything) | Any measurable metric: prompts, API perf, landing pages, test suites, SQL |
| [jmilinovich/goal-md](https://github.com/jmilinovich/goal-md) | GOAL.md pattern where agents construct fitness functions before optimizing |
| [james-s-tayler/lazy-developer](https://github.com/james-s-tayler/lazy-developer) | Orchestrates autoresearch across prioritized optimization goals |
| [Entrpi/autoresearch-everywhere](https://github.com/Entrpi/autoresearch-everywhere) | Cross-platform auto-detection and loop startup |
| [greyhaven-ai/autocontext](https://github.com/greyhaven-ai/autocontext) | Closed-loop control plane with evaluation, persistent knowledge, staged validation |

### Research Agent Systems
End-to-end autonomous research pipelines:

| Tool | Description |
|------|-------------|
| [SakanaAI/AI-Scientist](https://github.com/SakanaAI/AI-Scientist) | First comprehensive system for fully automatic scientific discovery |
| [SakanaAI/AI-Scientist-v2](https://github.com/SakanaAI/AI-Scientist-v2) | Workshop-level discovery via agentic tree search |
| [HKUDS/AI-Researcher](https://github.com/HKUDS/AI-Researcher) | NeurIPS 2025 - Full automation: hypothesis to manuscript to peer review |
| [OpenRaiser/NanoResearch](https://github.com/OpenRaiser/NanoResearch) | Plans experiments, generates code, runs jobs, writes papers |
| [SamuelSchmidgall/AgentLaboratory](https://github.com/SamuelSchmidgall/AgentLaboratory) | End-to-end: idea to literature review to experiments to report |
